本專案為一套以光學劃記辨識（Optical Mark Recognition, OMR）為基礎之自動閱卷系統，透過影像處理技術自動辨識學生於答案卡上填塗之選項，並與標準答案進行比對後完成計分。系統不需仰賴昂貴之專用閱卷機設備，僅需一般掃描器即可使用，適合中小型課程、教師測驗以及學術研究用途。

本系統主要分為兩個子模組。第一個為答案卡產生模組，用於將學生名單資料（Excel）與既有答案卡模板結合，自動產生每位學生專屬的答案卡。答案卡內容包含學生姓名與 QR Code，其中 QR Code 可作為學生身分識別資訊，輸出格式為可列印之 PDF 檔案。第二個為 OMR 自動閱卷模組，負責讀取學生作答掃描檔（PDF），透過影像處理與圓形偵測技術辨識填塗選項，並與標準答案進行比對後完成自動計分，最後輸出學生作答結果與成績資料。

在影像處理流程中，系統會先將學生作答 PDF 轉換為影像，再擷取答案區域（ROI），建立答案泡泡的位置配置，並根據圓形區域內的黑色像素比例判斷是否填塗。系統同時支援自動偵測與手動設定答案位置，以提升在不同掃描品質與答案卡格式下的辨識穩定度。

本系統支援單選題與多選題。對於多選題，使用者可選擇全對給分或部分給分機制，以提升測驗評量的彈性。在掃描解析度達 300 DPI 且影像歪斜角度較小的情況下，系統可穩定辨識大多數作答選項，並正確完成計分。

整體而言，本專案結合 Python 程式設計、影像處理與圖形化操作介面，實作一套具備實用性與延展性的自動閱卷系統，能有效降低人工閱卷成本，並提升評分效率與準確性，適合作為課程期末專題與教學應用之參考。
